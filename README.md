# TalkBridge

Learn together. Build a shared vocabulary. Practice naturally.

TalkBridge is an open-source web application designed for language exchange partners, tutors, friends, and couples who are learning each other's languages.

Instead of sending vocabulary back and forth through chat, users share a room where they can build a collaborative vocabulary list that stays available whenever they meet again.

## Features

### Shared Study Rooms
- Create a room in seconds
- Share a simple URL
- Join without complicated setup

### Shared Vocabulary
- Add words together
- Store translations
- Keep learning across multiple sessions

### Designed for Conversation
Instead of traditional flashcards, TalkBridge is built around real conversations.

When someone doesn't know a word, add it immediately and continue talking.

## Getting Started

Clone the repository.

```bash
git clone https://github.com/zuisetu/talkbridge.git
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## Why TalkBridge?

Most language learners use messaging apps to exchange vocabulary.

Those words quickly disappear into chat history.

TalkBridge provides a shared study space where both learners continue building vocabulary over time.

The goal is to make language exchange feel effortless.
